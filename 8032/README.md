# 8032: API (RESTFUL Node.js app)
- Nodejs REST API
- Token based authentication
- Angularjs client

# Dependencies
- Node v7.6.0

# Get started
- NPM init
```
npm init
```
- Dependencies
```
npm install --save express body-parser morgan jwt-simple
```

# Development
- Login
```
http://localhost:3000/login
```

# API
```
/login
```
